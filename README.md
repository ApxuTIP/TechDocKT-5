# Навыки, приобретенные в рамках курса технической документации

## **Инструменты для разработки документации**
- **GitHub** - система контроля версий для работы над документацией
- **Markdown** - язык разметки для создания читабельной и структурированной документации

- Проектирование сценариев и функциональных требований
